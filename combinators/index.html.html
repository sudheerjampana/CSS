Combinators and Selectors

In CSS, combinator selectors define relationships between two or more elements. They are used to select elements based on their position or relationship in the HTML structure.

Types of Combinator Selectors:

  1. Descendant Selectors
  2. Child Selectors
  3. Adjacent Sibling Selectors
  4. General Sibling Selectors

1.Descendant Selectors

  • It selects the all the child, grand child elements
  • It is represented with space(" ")

Example:

div p { color: blue; }

2.Child Selectors

  • It selects only child elements from the main selected parent
  • It is represented with space(">")

Example:

div > p { color: red; }

3.Adjacent Sibling Selectors

  • It selects the next immediate elements from the selected
  • It is represented with space("+")

Example:

div+p { color: green; }

4.General Sibling Selectors

  • It selects all elements that are siblings after a specified element.
  • It is represented with space("~")

Example:

div~p { color: yellow; }
